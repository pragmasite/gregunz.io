• 77 171 Qui est responsable d'un bateau par rapport à l'autorité d'admission?
    • Plusieurs personnes peuvent être responsables, toutes doivent être mentionnées dans le permis de navigation
    • Lorsque plusieurs personnes sont détentrices d'un bateau, elles désignent aux autorités d'admission le représentant responsable qui est inscrit dans le permis de navigation
    • Le détenteur inscrit sur le permis de navigation
